# 🔍 SEO Keyword Studio

A lightweight, no-fluff keyword brainstorming tool built for SEO workflows in the **hotel and apartment/rental industry**.

Ever sit down to do keyword research and your brain just... goes blank? This fixes that.

👉 **[Try it live](http://seo-key-word-search.vercel.app/)**

---

## What it does

You pick an industry, type in a city (and optionally a neighbourhood), select the page type you're working on, and instantly get a organised set of keyword ideas — ready to copy straight into your workflow.

Keywords are grouped into three categories to make it easy to pick the right ones:

- **Broad / Head Terms** — high-volume, general keywords
- **Intent-Based** — action and decision-stage keywords
- **Location-Specific** — neighbourhood-level keywords (shown when you add an area)

## How to use it

**Option 1 — Use the live site**
Just visit [seo-key-word-search.vercel.app](http://seo-key-word-search.vercel.app/) in your browser. Nothing to install.

**Option 2 — Run it locally**
1. Fork or clone this repo
2. Open `index.html` in your browser
3. That's it — no dependencies, no build step, no backend

---

## Copy options

- **Copy individual keywords** — one at a time
- **Copy Group** — grabs all keywords in a category
- **Copy All** — everything at once, one keyword per line, ready to paste into Excel or any SEO tool

---

## Tech

Pure HTML, CSS, and JavaScript. Single file. No frameworks, no backend, no tracking. Works offline once loaded.
